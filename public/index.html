REST servis for JWT by Aliaksandr Babrouski

Create User in Mongo DB:
"POST" query to "https://koa-jwt.herokuapp.com/user"
with body:
{
"displayName": "John",
"email": "Doe@gmail.com",
"password": "123456"
}
Get a JWT:
"POST" query to "https://koa-jwt.herokuapp.com/login"
with body:
{
"email": "Doe@gmail.com",
"password": "123456"
}
Authentication with "JWT" key:
"GET" query to "https://koa-jwt.herokuapp.com/custom"

headers: {
'Authorization': 'JWT xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'
}
if correct, you get "displayName"
